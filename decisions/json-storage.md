# Decision: Use JSON for data storage

## Question
How should data be stored in the Hotel Management System?

## Context
The application is a small desktop system and does not require a full database setup.

## Alternatives
- SQL database
- JSON files

## Decision
Use JSON files.

## Reasoning
JSON is simple, lightweight, and sufficient for the current scope of the project.

## Expected Outcome
Easy local persistence with low setup complexity.
