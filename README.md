# Software Engineering Knowledge Base

## Purpose
This repository documents my learning process in the Software Engineering course.

It is not only a collection of notes, but a structured environment for:
- tracking insights
- documenting decisions
- recording experiments
- reflecting on learning over time

## Repository Structure
- /beliefs - concepts and things I believe to be true based on lectures and study
- /decisions - development or learning decisions and their reasoning
- /experiments - practical attempts, tests, and observations
- /notes - lecture notes and summaries

## Rules for Using This Repository
- I separate beliefs, decisions, and experiments
- I document reasoning, not only conclusions
- I update the repository during the course
- I use Git commits to track the evolution of my learning
